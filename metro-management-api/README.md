# Metro Management System API

A backend RESTful API for managing metro rail operations including users, stations, routes, trains, ticket booking, payments, trip history, and email notifications.

## Tech Stack

- NestJS
- TypeScript
- PostgreSQL
- TypeORM
- JWT Authentication
- bcryptjs
- Nodemailer
- Handlebars
- Postman

## Features

- User registration and login
- JWT authentication
- Role-based access: admin, staff, passenger
- User profile update
- Change password
- Forgot/reset password
- Station management
- Route and RouteStation management
- Train management
- Ticket booking and cancellation
- Payment processing
- Trip history
- Email notifications for ticket booking, payment success, and ticket cancellation
